HTML TAGS
  • "title" tag: The "title" tag is used to define the title of an HTML document. It appears in the browser's title bar or tab and helps identify the webpage. It provides a concise description or name for the document
  • "body" tag: The "body" tag represents the main content area of an HTML document. It contains all the visible elements that users see when they visit a webpage, such as text, images, links, and other types of content.
  • "header" tag: The "header" tag represents a section at the top of a webpage. It typically contains introductory or identifying information about the webpage or website, such as logos, navigation menus, or page titles.
  • "p" tag: The "p" tag represents a paragraph of text within an HTML document. It is used to separate and structure blocks of textual content, providing visual spacing and indicating that a new paragraph begins after the opening "p" tag and ends before the closing "p" tag.
  • "footer" tag: The "footer" tag represents a section at the bottom of a webpage. It often includes information like copyright notices, contact details, links to related pages, or other supplementary content that appears consistently across the webpage or website.
  • "img" tag: The "img" tag is used to insert an image into an HTML document. It specifies the source file of the image using the "src" attribute and allows you to customize the image's size, alt text, and other properties.
